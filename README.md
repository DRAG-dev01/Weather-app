\# Weather App



A desktop app that fetches live weather data for any city.



\## Features

\- Real-time weather via wttr.in API

\- City validation using fuzzy matching

\- Error handling for connection issues



\## How to run

1\. Install dependencies: `pip install requests`

2\. Run: `python weather\_app.py`



\## Built with

\- Python

\- tkinter

\- requests

\- difflib
